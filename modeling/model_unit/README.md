# **卷积结构单元**

模型零件表示一个个基本的结构单元，目前支持的结构单元有：

| 左对齐标题 | 右对齐标题 | 居中对齐标题 |
| :------:| :------: | :------: |
| 短文本 | 中等文本 | 稍微长一点的文本 |
| 稍微长一点的文本 | 短文本 | 中等文本 |
